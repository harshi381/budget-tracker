Developed a full-stack personal finance tracker with a responsive user interface, enabling users to log 
expenses, view totals, and manage their spending effectively. 
• Integrated a custom, secure calculator to assist with expense calculations & Incorporated a third-party API 
(Frankfurter API) to fetch real-time currency exchange rates, allowing for dynamic currency conversion and 
accurate expense tracking across multiple currencies . 
•  Designed a clean and modern UI with a split-panel layout, custom button styles, and responsive design 
principles to ensure a seamless user experience. 
• Tools and Technologies Used : Frontend- HTML5, CSS3, JavaScript, Backend- Firebase (Authentication, 
Firestore Database) ,APIs- Frankfurter API
